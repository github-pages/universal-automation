# Remove-UASchedule

## SYNOPSIS
Removes a schedule for a script.

## SYNTAX

```
Remove-UASchedule [-Schedule] <Schedule> [-ComputerName <String>] [-AppToken <String>] [<CommonParameters>]
```

## DESCRIPTION
Removes a schedule for a script.

## EXAMPLES

### Example 1
```powershell
PS C:\> $Schedule = Get-UASchedule -Id 12
PS C:\> Remove-UASchedule -Schedule $Schedule
```

Removes schedule 12.
