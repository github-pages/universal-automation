# Stop-UAJob

## SYNOPSIS
Stops a UA job.

## SYNTAX

```
Stop-UAJob [-Job] <Job> [-ComputerName <String>] [-AppToken <String>] [<CommonParameters>]
```

## DESCRIPTION
Stops a UA job.

## EXAMPLES

### Example 1
```powershell
PS C:\> $Job = Get-UAJob -Id 12
PS C:\> Stop-UAJob -Job $Job
```

Stops job 12.
